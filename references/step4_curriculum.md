# STEP 4 参照資料: カリキュラムの設計（6ヶ月分）

STEP4に進んだ時点でこのファイルを読み込む。ユーザーのジャンルに応じて6章構成のカリキュラムを提案する。

---

## カリキュラム設計の原則

- **6章構成**（各章に3つの講義項目）
- **実践順**で並べる（基礎→応用→実践→仕組み化）
- **難易度**を段階的に上げる
- **受講体験の流れ**を意識する（小さな成功体験→自信→大きな挑戦）

---

## 構造テンプレート（インスタ集客講座の事例）

以下は「0→100万を完全攻略するインスタ集客マスター講座」の構成例。他ジャンル（ダイエット／婚活／キャリア支援など）に応用する際は、**構造をそのまま使い、内容を各分野に変換する**。

| 章 | タイトル | 講義項目 |
|---|---|---|
| 第1章 | 0から始めるコンセプトメイキング | 状態ベースで定めるターゲット設定 / 共感と未来が伝わる理想のゴール設計 / USP（独自の強み・価値）の言語化フレーム |
| 第2章 | 実績なしから高単価で売れる商品設計 | Before→Afterで設計する講座構成テンプレート / 受講生が変化を感じる5STEP設計 / 高単価で選ばれる商品の魅せ方と言語化 |
| 第3章 | インスタグラム集客基礎（設計＋リール基礎） | フォローされるプロフィールと世界観設計 / 伸びるリール台本の基本構成とテーマ選定 / 保存・共感されるリールの作り方 |
| 第4章 | リール台本の作成と投稿ルーティンの仕組み化 | 台本構成テンプレートの活用と再現 / バズる動画編集5STEP / 投稿スケジュールとリール運用の仕組み化 |
| 第5章 | LINE導線とステップ設計 | リールから自然に誘導するプロフィール設計 / LINE登録後のステップ配信テンプレ7選 / ストーリー×LINEで信頼を深める導線設計 |
| 第6章 | セールスとバックエンド設計 | セールス前提のストーリー設計と事前教育 / 自然に申込が入る無料講座・診断コンテンツの構成 / 売上が継続するバックエンド商品の設計 |

---

## ジャンル変換のガイドライン

上記テンプレートをユーザーのジャンルに変換する際の対応表：

| テンプレートの章 | 汎用的な役割 | 変換のポイント |
|---|---|---|
| 第1章 | コンセプト・ターゲット設計 | どのジャンルでもそのまま使える基礎 |
| 第2章 | 商品設計・構成 | ジャンル固有の成果指標に変換 |
| 第3章 | 集客の基礎スキル | ジャンルの主要集客チャネルに変換 |
| 第4章 | コンテンツ制作の仕組み化 | ジャンル固有のコンテンツ形式に変換 |
| 第5章 | 見込み客の育成・導線設計 | ジャンルに合った信頼構築手法に変換 |
| 第6章 | セールス・収益化 | ジャンル固有の販売手法に変換 |

### 変換例（ダイエットコーチの場合）

| 章 | タイトル例 |
|---|---|
| 第1章 | 理想のボディを明確にするゴール設計 |
| 第2章 | リバウンドしない食事×運動プログラム設計 |
| 第3章 | SNSで「この人に教わりたい」と思わせるビフォーアフター発信 |

| 第5章 | 無料体験セッションへの導線設計 |
| 第6章 | 継続コースとリピート設計 |
