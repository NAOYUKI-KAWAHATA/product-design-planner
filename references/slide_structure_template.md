## レイアウト種別の定義

| 種別 | 説明 |
|---|---|
| cover | 表紙。タイトル＋サブタイトル＋背景画像指示 |
| section-title | セクション扉。大見出し＋装飾 |
| statement | 1文インパクト。大きな文字で1メッセージのみ |
| bullet-list | 箇条書きリスト（3〜9項目） |
| profile | 講師プロフィール。写真＋実績リスト |
| two-column | 2カラム構成（章一覧など左右分割） |
| phase-card | フェーズ説明。番号＋タイトル＋説明文＋画像指示 |
| feature-card | 提供コンテンツ紹介。タイトル＋説明＋サンプル映像枠 |
| comparison | 比較表（自社 vs 他社） |
| pain-point | 悩み・不安の提示（共感スライド） |
| pricing | 価格提示。プラン名＋価格＋提供内容 |
| bonus | 特典紹介。特典名＋説明＋画像指示 |
| diagram | 図解・フロー（仕組みの説明） |
| social-proof | 実績・写真ギャラリー（イベント写真等） |
| contrast | 対比スライド（左右で比較） |
| cta | エンディング・行動喚起 |
| payment | 支払い方法の案内 |
